# LimeSDR Mini v2

LimeSDR Mini 2.0 is an upgraded, drop-in replacement for LimeSDR Mini, a hardware platform for developing and prototyping high-performance and logic-intensive digital and RF designs that use an FPGA coupled with Lime Microsystemsâ€™ LMS7002M RF transceiver.

## Contents

The directory structure is as follows:

      hardware/<version>/
          OutputJobs/            - Output jobs
          PCB/                   - PCB design
          Project Outputs/       - BOM, rule check reports, Gerbers, pick & place files, PDFs
          Schematics/            - Schematic diagrams
